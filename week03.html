Overview

Week 03 covered interaction styles and interface types. Interaction styles define how users communicate with computer systems.

According to Dix et al., choosing the right interaction style is crucial for usability and efficiency, depending on user expertise and task type (Chapter 4).

Interaction Styles

  • Command-line interfaces
  • Menu-based interfaces
  • Form-based interfaces
  • Direct manipulation interfaces

Reflection

Understanding interaction styles helps in designing interfaces that match user expectations, reduce errors, and improve satisfaction. For example, direct manipulation is often better for novice users than command-line systems.
